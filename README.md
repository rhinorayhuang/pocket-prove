# Pocket Grove

Pocket Grove 是一個以 Vue 3、TypeScript、Vite 製作的原創 GB 風怪獸 RPG 原型。

目前已完成的可玩內容：
1. 世界地圖移動與場景切換
2. NPC 對話與路牌互動
3. 草叢野生遭遇戰
4. 晨芽鎮外圍、晨芽鎮廣場、晨芽研究所、曦徑一號路
5. 第一場訓練家戰
6. `engine / scenes / systems / data / types` 分層骨架

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

建置輸出會在 `dist/`。
